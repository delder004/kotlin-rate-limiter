# Code of Conduct

## Our Standard

This project is committed to providing a respectful, welcoming, and harassment-free experience for everyone participating in the community.

Examples of expected behavior:

- Be respectful in discussions and code review.
- Assume good intent, but be precise and constructive.
- Accept feedback gracefully.
- Focus on what is best for the project and its users.

Examples of unacceptable behavior:

- Harassment, discrimination, or intimidation
- Personal attacks or insulting language
- Trolling, deliberate disruption, or bad-faith engagement
- Publishing private information without permission

## Scope

This code of conduct applies to project spaces, including issues, pull requests, discussions, and other public interactions around the repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that do not align with this code of conduct.

## Reporting

To report unacceptable behavior, contact the maintainer through GitHub: [@delder004](https://github.com/delder004).

All reports will be reviewed and handled as promptly and fairly as possible.
